# BppClient

Ruby client for handling the Brasil pre-pagos API

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'bpp_client'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install bpp_client
